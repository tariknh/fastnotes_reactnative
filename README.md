# FastNotes (Expo / React Native)

## Dev setup

**Prereqs**: Node.js (LTS) + npm.

```bash
npm install
npx expo start
```

Then run it on a target:

- Press `i` for iOS Simulator (requires Xcode)
- Press `a` for Android emulator (requires Android Studio)
- Scan QR with **Expo Go** on a phone
- Press `w` for web

Start editing in the `app/` directory (Expo Router).
